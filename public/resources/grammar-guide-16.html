Unit 16: Object Pronouns

Direct (OD) and Indirect (OI) Objects

1. Grammar: Pronouns
2. Practice: Exercises

1. Direct vs. Indirect Object

Direct Object (OD)

Receives the action directly. Answers "What?" or "Who?"

  • Me (me)
  • Te (you)
  • Lo / La (him, her, it)
  • Nos (us)
  • Os (you all)
  • Los / Las (them)

Indirect Object (OI)

Receives the action indirectly. Answers "To whom?" or "For whom?"

  • Me (to me)
  • Te (to you)
  • Le (to him, her, it)
  • Nos (to us)
  • Os (to you all)
  • Les (to them)

Placement Rules:

  • Before conjugated verb: Lo compro.
  • Attached to Infinitive/Gerund: Voy a comprarlo. / Estoy comprándolo.
  • Order: Indirect comes first! OI + OD (Me lo das).
  • The "Se" Rule: If both start with 'L' (le lo), change 'Le' to 'Se'.
    Le lo doy -> Se lo doy.

2. Exercises

Replace the underlined words with the correct pronoun.

1 Exercise: Direct Object (Lo, La, Los, Las)

Example: Compro el pan -> Lo compro.

2 Exercise: Indirect Object (Le, Les)

Example: Escribo a Juan -> Le escribo.

3 Exercise: Double Pronouns (OI + OD)

Remember: People first! (And Le->Se if needed).
Ex: Doy el libro a Ana -> Se lo doy.
